""professional"" opinions

I care about automated validation, constraint solving in game applications; and, again, programming languages for game development.

There's a theme here: I firmly believe that we can and should use tooling improvements to take the grunt work off of the shoulders of devs. Unit testing, automated linting, bots that crawl over source code and apply systematic improvements, etc. We made the computers good at that kind of stuff for a reason.
